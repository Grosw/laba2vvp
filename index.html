Команды Git
Команда Описание
git init Инициализация. Создает новый пустой Git-репозиторий в текущей папке. Появляется скрытая директория .git, где будет храниться вся история изменений.
git clone url Клонирование. Копирует существующий удаленный репозиторий по указанному адресу (url) на ваш компьютер.
git add file Подготовка файла. Добавляет конкретный файл в "индекс". Это значит, что изменения в этом файле будут включены в следующий коммит.
git add . Подготовка всех изменений. Добавляет все новые, измененные и удаленные файлы в текущей директории (и поддиректориях) в индекс для коммита.
git commit -m "comment" Фиксация изменений. Сохраняет все подготовленные изменения (из индекса) в историю репозитория. Флаг -m позволяет сразу написать комментарий к версии.
git commit -a -m "comment" Быстрый коммит. Автоматически добавляет в индекс все измененные и удаленные файлы (которые уже отслеживались Git) и сразу создает коммит. Важно: новые файлы (untracked) этой командой не подхватятся.
git push url Отправка данных. Загружает ваши локальные коммиты в удаленный репозиторий по указанному адресу.
git status Состояние. Показывает текущее состояние рабочего дерева: какие файлы изменены, какие подготовлены к коммиту, а какие Git еще не отслеживает.
git log История. Выводит список всех коммитов, сделанных в текущей ветке, с именами авторов, датами и комментариями.
git branch branch_name Создание ветки. Создает новую ветку с именем branch_name, но не переключает на нее.
git checkout branch_name Переключение. Переходит на указанную ветку (или восстанавливает файлы из нее в рабочую директорию).
git checkout -b branch_name Создание и переход. Комбинированная команда: создает новую ветку и сразу переключает вас на нее.
git remote Удаленные репозитории. Показывает список имен подключенных удаленных репозиториев (например, origin).
